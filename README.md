# CyberTech RH - HR-HUB PLUS 🚀

Sistema completo de Gestão de Recursos Humanos com design premium e funcionalidades administrativas avançadas.

## 🛠️ Funcionalidades Principais

- **Dashboard Executivo**: Métricas de gênero, custo, cargos e CIDs.
- **Gestão de Funcionários**: Cadastro, edição, inativação e controle de salários.
- **Atestados Médicos**: Lançamento, monitoramento de dias e arquivos anexos.
- **Folha de Pagamento**: Resumo mensal, descontos, faltas e cálculos automáticos.
- **Prestadores de Serviços**: Gestão de contratos externos com controle de custos e alertas.
- **Rescisões**: Módulo para desligamentos com seletor de motivos (Pedido, Justa Causa, Acordo, etc).
- **Audit Logs**: Histórico completo de ações administrativas para auditoria.
- **Configurações Admin**: Painel para configuração de IP, Porta e URLs de conexão remota.

## 🎨 Design e Tech Stack

- **Framework**: Vite + React + TypeScript
- **Estética**: Glassmorphism & Dark Mode Premium (CyberTech RH Standard)
- **Ícones**: Lucide React
- **Deploy**: Otimizado para Vercel (inclui `vercel.json` e `.npmrc`)

## 🚀 Como Rodar Localmente

1.  Instale as dependências:
    ```bash
    npm install
    ```
2.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
3.  Acesse: `http://localhost:8080`

## 📡 Como Fazer o Deploy

O sistema já está pronto para Vercel. Basta subir para o GitHub e conectar o repositório.
